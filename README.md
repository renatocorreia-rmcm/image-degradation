# Degradação de imagens.

Degradação de imagens devido ao problema de representação em ponto flutuante: caso de magnificação/contração/deformação de imagens ou cópias repetidas.

Deve-se implementar uma ilustração do que ocorre com uma imagem quando ela é transformada através de operações que são sensíveis à representação numa máquina pequena, cujos parâmetros são dados.
